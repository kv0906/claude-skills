# Claude Skills

Personal curated skills for Claude Code.

## Skills

### premium-frontend-skill

Create award-winning, cinematic frontend interfaces that feel alive. Specializes in:
- WebGL & custom shaders (CPPN, holographic, digital-liquid)
- Premium animations with GSAP/Framer Motion
- Glassmorphism, dark themes, bento layouts
- Templates for dashboards, hero sections, preloaders

**Trigger:** Use when building landing pages, dashboards, or any interface where "generic AI slop" is unacceptable.

### will-writing

Ghost writer for Telegram channel "Will's Thoughts". Vietnamese content with tech philosopher style:
- AI + Buddhism + Philosophy synthesis
- Peer-level voice ("ae", "mình")
- Cross-domain thinking (Tech ↔ Eastern wisdom)

**Trigger:** "viết cho tôi", "will skill", "viết post telegram"

## Usage

Add to Claude Code settings:
```json
{
  "skills": [
    "/path/to/claude-skills/premium-frontend-skill",
    "/path/to/claude-skills/will-writing"
  ]
}
```
